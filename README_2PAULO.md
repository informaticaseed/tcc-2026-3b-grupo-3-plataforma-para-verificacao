## Sobre o Projeto

Este projeto tem como objetivo desenvolver uma plataforma para analisar arquivos e URLs suspeitas, ajudando a identificar possíveis malwares e ataques de phishing.

## Tecnologias Utilizadas

- Python
- FastAPI
- HTML
- SQL

## Funcionalidades

- Análise de links suspeitos.
- Verificação de arquivos.
- Armazenamento das análises em banco de dados.
- Interface simples para o usuário.
